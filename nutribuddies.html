NutriBuddies

NutriBuddies is a fun and interactive nutrition application made for kids and parents. This app highlights the importance of nutrition while also providing children with fun interactive games and quizzes.

About this project

January - February 2023 (8 weeks)

Student Designer, User Research, Finalizing Design

NutriBuddies strives to be a resource for nutritional information for children and parents. With interactive elements, children are more engaged with this family friendly application.

Skills used for this project

Figma

Figma is a collaborative interface design tool that allows multiple users to create, edit, and share designs in real-time.

User Research

User research is essential for understanding the needs, behaviors, and preferences of individuals who will interact with a product or service.

Wireframing

Wireframing is a crucial step in the design process, enabling designers to create skeletal outlines of digital interfaces that focus on layout, functionality, and user flow before diving into visual design elements.

Google Forms

Google Forms is a user-friendly, web-based tool that enables effortless creation of customizable surveys and forms, with data collection and analysis capabilities.

Google Sheets

Google Sheets is a versatile cloud-based spreadsheet application that facilitates collaborative data management, analysis, and visualization.
