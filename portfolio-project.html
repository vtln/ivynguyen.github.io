About this project

November - December 2023 (3 weeks)

Student Designer & Coder

The objective of this project was to have creative freedom in creating a website consisting of 5 HTML pages and utilizing CSS.

Skills used for this project

HTML

HTML is a markup language used for structuring and presenting content on the web.

CSS

CSS is a stylesheet language that defines the visual presentation of HTML elements.

DreamWeaver

Dreamweaver is a web design and development tool that provides a visual interface for building websites and editing code.

Wireframing

Wireframing is a crucial step in the design process, enabling designers to create skeletal outlines of digital interfaces that focus on layout, functionality, and user flow before diving into visual design elements.
